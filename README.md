# ClockTower

The Clock Tower is an Escape Room, set in a broken Victorian Clock Tower

Story: Time has stopped, the player must travel through the levels of the tower, repairing the system in order to escape. 

Takes about 10 minutes to complete the game

3 levels: Keeper’s Quarters > Clock Room > Belfry
